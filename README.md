# C Language Projects

A collection of C programming exercises and projects covering fundamental concepts including algorithms, data structures, and practical applications.

## Projects

- **algoritmo_luhn** - Luhn algorithm implementation for credit card validation
- **algoritmo_luhn_tarjetas_credito** - Credit card validation using Luhn algorithm
- **caracteresV2.0** - Character manipulation utilities
- **AREA_CIRCULO** - Circle area calculation
- **HOTEL** - Hotel management system
- **DIA_SEMANA_strcmp_if** - Day of week determination
- **matriz** - Matrix operations
- **PRINTF_SCANF** - Input/Output examples
- **tarjetas_ia** - Credit card analysis with AI

## Building

Most projects include a Makefile. To build:

```bash
make
```

Or compile manually:

```bash
gcc -o output_name source.c
```

## Requirements

- GCC compiler
- Make (for projects with Makefile)
